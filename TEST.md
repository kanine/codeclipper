# Run and Test

1. Press `F5` on your keyboard. This opens a new VS Code window, the Extension Development Host, with your extension loaded.
2. In that new window, open any file.
3. Highlight a few lines of code.
4. Right-click and you should see `Copy File and Line Reference` at the top.
5. Click it, then try pasting (`Ctrl+V`) into a text file or chat.
